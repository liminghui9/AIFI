============================================
  AIFI 智能财报系统 - 快速开始指南
============================================

一、首次使用（三步走）

步骤1：安装依赖
   双击运行: install.bat
   或命令行: pip install -r requirements.txt

步骤2：启动系统
   双击运行: start.bat
   或命令行: python app.py

步骤3：打开浏览器
   访问: http://localhost:5000

二、使用系统

1. 准备测试数据
   - 使用 sample_data/sample_template.xlsx（需要先生成）
   - 或参考格式准备自己的Excel文件

2. 生成示例数据（可选）
   cd sample_data
   python create_sample_excel.py

3. 上传Excel文件
   - 在浏览器中点击上传按钮
   - 选择Excel文件（.xlsx 或 .xls）
   - 等待系统分析

4. 查看和导出报告
   - 自动跳转到报告页面
   - 点击"导出Word"或"导出PDF"

三、配置AI功能（可选）

如需启用AI智能分析：

1. 创建 .env 文件（在项目根目录）
2. 添加以下内容：
   OPENAI_API_KEY=your_api_key_here
   OPENAI_API_BASE=https://api.openai.com/v1

注意：不配置也可使用，系统会使用默认分析逻辑

四、Excel数据格式要求

基本信息字段：
- 企业名称
- 统一社会信用代码
- 注册资本（万元）
- 成立日期
- 行业类别
- 法定代表人

财务数据字段（需要两年数据）：
- 格式：字段名_年份
- 例如：总资产_2023、总资产_2022

必需的财务字段：
资产负债表：总资产、流动资产、非流动资产、总负债、流动负债、非流动负债、所有者权益
利润表：营业收入、营业成本、营业利润、利润总额、净利润
现金流量表：经营活动现金流量净额、投资活动现金流量净额、筹资活动现金流量净额、现金及现金等价物净增加额

五、常见问题

Q: 端口5000被占用？
A: 修改 app.py 最后一行的端口号

Q: 找不到模块？
A: 重新运行 install.bat 安装依赖

Q: 数据缺失提示？
A: 检查Excel字段名和格式，参考示例文件

Q: PDF中文显示异常？
A: 建议使用Word导出
